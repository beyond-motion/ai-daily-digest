### 2. GPT-5.6 三档发布：Luna / Terra / Sol

7/9 OpenAI 发布 GPT-5.6 系列旗舰，三个尺寸分档定价（每百万 token input/output）：
- **Luna**：$1 / $6（小）
- **Terra**：$2.50 / $15（中）
- **Sol**：$5 / $30（大）

参数对比：Claude Opus $5/$25，Claude Fable 5 $10/$50。但 token 价格对比意义有限，因为推理 token 数量差异大。

三个模型共同点：2026-02-16 知识截止、**100 万 token 上下文**、最大输出 12.8 万 token。

**关键后果**：Simon Willison 一句话定性——"GPT-5.6 Sol 是 Fable/Mythos 级别模型"——直接迫使 Anthropic 第三次推迟 Fable 5 的下线日期，从 7/5 → **7/19**，并保留 Claude Code 50% 更高的每周速率限制。这是上周 Sonnet 5 涨价 30% 之后又一次重大调整。

🔗 https://simonwillison.net/2026/Jul/9/gpt-5-6/

### 3. ChatGPT Mac 应用大重写：Benedict Evans 评 "What a total mess"

7/9 OpenAI 把桌面端彻底重组：
- 旧 ChatGPT 应用改名为 **ChatGPT Classic**
- **Codex** 改名为新的 **ChatGPT**（含 Codex 图标选项）
- 引入新概念 **ChatGPT Work**（云端办公）和 **ChatGPT Codex**（桌面技术工作）
