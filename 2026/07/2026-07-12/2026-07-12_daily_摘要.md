## 🤖💬 OpenAI 新版 ChatGPT 桌面应用："超级应用"还是"超级混乱"？

**来源**：Daring Fireball（John Gruber）· 2026-07-11

OpenAI 本周推出了全新"超级"ChatGPT 桌面应用，整合了 Chat、Work 和 Codex 三大模式。然而，首批用户反馈和资深评论者的体验几乎**一致负面**。

### Gruber 的三连批评

**① 命名混乱**：旧版应用被保留为"ChatGPT Classic"，但新版应用既不通过内置更新提示用户升级，手动安装后又与旧版产生冲突——在已运行旧版的情况下直接打开磁盘镜像，应用会"弹来弹去"无法正常使用。Gruber 在三台 Mac 上测试，没有一台出现 OpenAI 帮助文档所描述的"应用内提示下载新版"流程。

**② 功能割裂**：OpenAI 自己的帮助文档暴露了产品设计的深层矛盾——
- Work 在网页/移动端运行在云端，桌面端则可访问本地文件，但**云端和桌面的 Work 对话互不同步**。
- Codex 仅在桌面端可用，移动端的"Remote"标签页可以查看桌面端任务，但**不会出现在移动端聊天历史中**。
- Project、Task、Chat 三个概念的区别不清，UI 中 floating window 行为不一致。

**③ 组织混乱映射到产品**：Gruber 引用 Benedict Evans 在 Threads 上的精辟评价——"Wow, what a total mess"。更犀利的是：

> "在内部政治主导的公司里，发布的产品反映了公司的组织架构图。OpenAI 的内部组织架构一团糟，这款新应用完美地反映了这一点。"

Gruber 还透露，据他了解，OpenAI 高层对 Anthropic 有一种**"FOMO 执念"**，公司内部真正有影响力的人都在关注竞争对手的一举一动，而非打磨自己的产品。旧版 ChatGPT 应用之所以专注、好用，是因为它目标单一。新版应用看似精致，UI 却只是**"穿了成人西装的草台班子"**。

🔗 https://daringfireball.net/2026/07/exactly_like_om_malik（含多个原始链接）

## 🗄️ sqlite-utils 4.1 发布：用 Python 代码直接生成数据行

**来源**：Simon Willison's Weblog · 2026-07-11 23:50

Simon Willison 发布了 sqlite-utils 4.1，这个版本虽然是 4.0 之后的小版本更新，但带来了几个实用的 CLI 增强：

**新功能一览**：
- `--code` 选项：`sqlite-utils insert` 和 `upsert` 现在可以接受一段 Python 代码（或 `.py` 文件路径），代码中定义 `rows()` 生成器函数即可直接插入数据，无需先准备 CSV/JSON 文件。
- `--type` 选项：可在建表时覆盖自动推断的列类型——解决邮编等"看起来是数字但应该是文本"的经典问题。
- `drop-index` 命令和方法：终于可以按名称删除索引了，支持 `--ignore` 忽略不存在的索引。
- SQL 查询支持从 stdin 读取：`sqlite-utils query data.db -` 现在可以从管道读取 SQL。

`--code` 功能是 Willison 一贯风格的延伸——他一直致力于让命令行工具既适合快速原型实验，又能在脚本中可靠使用。sqlite-utils 4.x 系列正在稳步成为 Python 生态中最顺手的 SQLite 操作工具。

🔗 https://simonwillison.net/2026/Jul/11/sqlite-utils/

## 🔢 Gilbreath 猜想新进展：一个简单到中学生能懂，却难得连 Erdős 都不敢证明的素数谜题

**来源**：John D. Cook · 2026-07-11 21:30

John D. Cook 撰文介绍了 Gilbreath 猜想的最新研究进展。

**什么是 Gilbreath 猜想？**
1. 写出素数序列：2, 3, 5, 7, 11, 13, ...
2. 计算相邻素数之差的绝对值：1, 2, 2, 4, 2, ...
3. 对差值序列再取相邻差：1, 0, 2, 2, ...
4. 重复这个过程……
5. 猜想：**每一行的第一个数字永远是 1**。

Paul Erdős 曾推测这个猜想是正确的，但也认为**现有数学工具不足以证明它**。Cook 的文章指向了近期在这一方向上的新突破，虽然完整证明仍然遥遥无期。

这类"简单到可以解释给任何人听，但难到没有人能证明"的问题，正是数学最迷人的一面。

🔗 https://www.johndcook.com/blog/2026/07/11/progress-on-gilbreaths-conjecture/
