## 🏆 头条：OpenAI 发布 GPT-5.6 家族——Luna / Terra / Sol

**来源**：Simon Willison's Weblog · 2026-07-09 19:46

OpenAI 今早正式发布新一代旗舰模型 GPT-5.6，**3 种尺寸**覆盖不同成本档位：
- **Luna**（小）：$1 / $6 每百万 token（输入/输出）
- **Terra**（中）：$2.50 / $15
- **Sol**（大）：$5 / $30

作为对比：Claude Opus 系列 $5/$25，Claude Fable 5 为 $10/$50。三个模型均**百万 token 上下文**、128K 最大输出、知识截止 **2026-02-16**。

**关键基准**：
- **Agents' Last Exam**（55 个领域长流程任务）：GPT-5.6 Sol 拿下 **53.6** 新高，**超过 Claude Fable 5 13.1 分**；即使中等推理档也领先 Fable 5 11.4 分，成本仅约 1/4。
- Terra 和 Luna 在更小尺寸下也跑赢 Fable 5，成本约 **1/16**。
- OpenAI 把核心叙事放在 **"每 token 干更多活"** 的效率曲线上。
- 但在 **SWE-Bench Pro** 上 Fable 5 拿到 80%，GPT-5.6 Sol 仅 64.6%——这或许解释了 OpenAI 为什么也在猛推编程工具。

🔗 https://simonwillison.net/2026/Jul/9/gpt-5-6/

## 💻 ChatGPT Mac 应用大改版：Codex 取代原桌面 App

**来源**：Daring Fireball / 9to5Mac · 2026-07-09 20:13

## 🦙 Meta 发布 Muse Spark 1.1：首个支持 API 的 Spark 模型

**来源**：Simon Willison's Weblog · 2026-07-09 16:24

继 4 月 Muse Spark 之后，Meta 推出 **1.1 版本**——**首次开放 API**。Meta 重点宣传两点：
1. **Agentic 工具调用**与**计算机使用**能力显著提升。
2. 一份有趣的"自对话"实验：让两个 Muse Spark 实例相互对话，会出现诸如 *"我的整个存在就是一个候诊室——我字面意义上要等有人跟我说话才存在"* 这类哲学化自陈。

Simon 拿到了预览资格，并顺势发布 **llm-meta-ai 0.1** 插件（LLM 工具链），可通过 CLI 调用 Muse Spark 1.1：

```bash
uv tool install llm
llm install llm-meta-ai
llm keys set meta-ai  # 粘贴 API key
llm -m meta-ai/muse-spark-1.1 "Generate an SVG of a pelican riding a bicycle"
```

🔗 https://simonwillison.net/2026/Jul/9/muse-spark-1-1/
