## 主题二：Apple 生态 · 图标之争、老将告别、工具兴衰

### 🎨 1. "Squircle Jail"——强制统一图标形状引发开发者众怒

Paul Kafasis（Rogue Amoeba）撰文 **"Free the Icons"**，John Siracusa 称之为"苹果对 Mac 开发者做过的最糟糕的设计决策"。

**问题**：Apple 自 OS 26（Liquid Glass）起强制所有 app 图标统一为"squircle"（方圆形），取消了原本百花齐放的图标轮廓。

**论点**：
- 形状是人类视觉系统最快识别的特征之一——交通标志就靠三角形/八边形区分
- 旧版 Keynote 图标的剪影比现在任何完整图标都更"iconic"
- 强制 squircle 等于**同时去掉了形状和颜色**两个区分维度（Clear/Tinted 模式进一步抹掉了色彩）
- Apple 自己的咖啡桌画册曾 celebrated 图标之美——现在的图标"一点不好玩"
- 如果 squircle 真的更好，让它在市场上赢，而不是用平台权力强推

> Gruber 类比：苹果给每个 app 戴上了" stupid unnecessary helmets"——你很难在一群戴同款头盔的人里认出谁是谁。

### 📝 2. Markdown 终于获得 Apple 官方 UTI

### 📦 3. Maestral——最优雅的开源 Dropbox 客户端——停止维护

Sam Schott 宣布 Maestral 不再活跃维护，项目将于 7 月 28 日归档。当前版本在证书过期前仍可使用。

**Maestral 的价值**：它恢复了 Dropbox 最初的理念——就是一个本地文件夹，安静地同步。不用 File Provider API，不搞"云占位文件"，CPU 和内存占用极低。

Gruber 深情悼念，并表达了对未来的迷茫："我不知道证书什么时候过期，也不知道接下来用什么。"

### 💾 4. Backblaze 不再备份 Dropbox/iCloud/OneDrive 文件

Backblaze 停止备份云存储服务中的文件——因为这些文件通过 File Provider API"看起来在那儿但实际不在本地"。

解法之一是用 Maestral（本地真实文件），但 Maestral 又恰好停止了维护。**两条路同时收窄**，Mac 用户的备份策略越来越尴尬。

## 主题四：数学之美 · e 的有理数近似 & 几何定理重现

### ∑ e ≈ 2721/1001：10-bit 分母、21-bit 精度

John D. Cook 发现了一个精妙的近似：

| 对比项 | 值 |
|--------|-----|
| e 的真实值 | 2.71828182… |
| 2721/1001 | 2.71828171… |
| 朴素近似 2718/1000 | 2.718 |

2721/1001 的分母是 10-bit 数，但近似精度达到 **21 bit**（几乎 8 位有效数字）。

在二进制下更震撼：

```
2721/1001 = 10.10110111111000010100…
e          = 10.10110111111000010101…
```

前 20 位完全一致，仅末位差 1。

### 📐 几何定理图重现

Cook 另一篇博文试图复现 Leon Bankoff（1964）的一个几何定理图：半圆内接三角形，CD 垂直于直径 AB，求证蓝色内切圆直径 = 绿色圆半径 + 橙色圆半径。

他发现"复现图表比定理本身更有趣"，特别是两个与圆弧相切的圆需要解二次方程。附带了完整的 Python matplotlib 代码。

### 🔧 FILE_FLAG_DELETE_ON_CLOSE 没有"后悔药"

Raymond Chen（The Old New Thing）讲解 Windows API：`FILE_FLAG_DELETE_ON_CLOSE` 标志一旦设置就不可撤销。
