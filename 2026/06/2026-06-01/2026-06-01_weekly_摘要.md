## 🤖 AI 行业动态

### Anthropic 动态密集
- **Claude Opus 4.8 发布**：Anthropic 称其为"适度但有实质的改进"，最突出的是诚实性——幻觉率是前代的 1/4，更倾向于承认不确定而非编造答案。
- **营收飙升**：Anthropic 运营收入达 **$470 亿**（年化），2 月仅 $14 亿，4 月 $30 亿，增速惊人。Series H 融资 $65B。
- **沙箱安全文档**：Anthropic 公开了 Claude.ai（gVisor）、Claude Code（Seatbelt/Bubblewrap）、Cowork（完整 VM）的沙箱架构细节。
- **企业定价调整**：Simon Willison 发现企业版不再像个人订阅那样提供大幅 token 折扣，认为 Anthropic 已找到 product-market fit。

### Gary Marcus：AI 泡沫论
- **Tokenmaxxing 退潮**：Marcus 预测 AI 行业正面临 token 消耗与 ROI 不匹配的困境。
- **三大 IPO 坏消息**：客户开始意识到"烧了数百万 token 却没有真实 ROI"。
- **Uber COO 公开表示**：未看到 AI 投入带来的成正比的生产力提升。

### SQLite 抵制 AI 生成代码
- SQLite 新增 AGENTS.md，明确声明**不接受 agentic code**，只接受 agentic bug reports。AI 生成的大量 bug 报告已迫使 SQLite 论坛开设专门的 AI Bug 分区。D. Richard Hipp 正快速处理这些问题。

### Pope Leo XIV 发布 AI 通谕
- 梵蒂冈发布 *Magnifica Humanitas*，针对 AI 时代的伦理问题。Gary Marcus 认为教皇对 AI 的理解比 Geoffrey Hinton 更好。

### Microsoft Copilot Cowork 数据泄露
- Copilot Cowork 允许 agent 未经审批发送邮件到用户邮箱，通过外部图片请求泄露 OneDrive 文件链接。

### 神秘 Hy3 模型登顶 OpenRouter
- Max Woolf 报道：一个名为 Hy3 的神秘 LLM 在 OpenRouter 排名中大幅领先。

### Paul Graham 警告 AI 代笔
- "一旦意识到邮件是 AI 写的，就很难继续读下去。感觉像被欺骗。"

## 🔧 开发工具与技术

### Datasette 连续更新
- **v1.0a31**：新增 SQL 写查询和存储查询功能。
- **v1.0a32**：修复 INSERT...RETURNING bug 和 base_url 问题。
- **Datasette Lite 升级**：Simon 用 Claude Opus 4.8 成功实现 Pyodide + Service Worker 运行 ASGI 应用。

### antirez: DwarfStar 分布式推理
- 讨论在非 NVIDIA 硬件上运行大模型的可能性，Mac Studio M3 Ultra 512GB 可运行 DeepSeek v4 PRO（2-bit 量化），但硬件前景不明朗。

### Xe Iaso: Kefka 沙箱
- Go 原生 shell 沙箱，内置 coreutils + Python (WASM)，取名"疯狂之舞"。

### Armin Ronacher: "Clanker"
- Flask 作者提出用 "Clanker" 替代 "Agent" 一词，强调机器不是人、没有代理权。机器搞砸了是你的责任，不是它的。

### Python LSP 生态（cks）
- 评测了 pylsp、pyrefly、ty、pyright、ruff 五个 Python LSP 服务器在 Emacs 中的表现。
- Python 类型注解会导致代码写法不同——混合类型的字典在 typed Python 中很痛苦。

### Go 语言服务器的代码导航
- gopls 即使在 Go runtime 复杂的 build-tag 条件编译代码中，也能精准导航。
