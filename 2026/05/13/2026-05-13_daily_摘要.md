## 1. 安全 | CopyFail：AI 发现潜伏 9 年的 Linux 提权漏洞

**CVE-2026-31431** 是一个存在于 Linux 内核 `algif_aead` 模块的高危提权漏洞，由安全研究员使用 AI 工具发现。该漏洞在 AF_ALG 加密接口中存在逻辑缺陷，可通过 732 字节的 Python 脚本将普通用户权限提升至 root。漏洞于 2017 年被引入，影响几乎所有主流 Linux 发行版（Ubuntu、Amazon Linux、RHEL、SUSE），CVSS 评分 7.8。

微软安全博客指出，该漏洞影响了"云上 Linux 工作负载的很大一部分以及数百万容器"。CERT-EU 已发布安全公告。Krebs 在本月的 Patch Tuesday 报道中特别提到，AI 平台不仅能帮助人类发现漏洞，还能通过自动化分析找到传统人工审计难以触及的深层逻辑缺陷。

**来源**: [Microsoft Security Blog](https://www.microsoft.com/en-us/security/blog/2026/05/01/cve-2026-31431-copy-fail-vulnerability-enables-linux-root-privilege-escalation/) | [Bugcrowd](https://www.bugcrowd.com/blog/what-we-know-about-copy-fail-cve-2026-31431/) | [CERT-EU](https://cert.europa.eu/publications/security-advisories/2026-005/)

## 2. AI/LLM | llm 0.32a2：推理模型 token 可视化

Simon Willison 发布了 `llm` 命令行工具 alpha 版本 **0.32a2**，最重要的新特性是支持推理模型的思考过程可视化。大多数具备推理能力的 OpenAI 模型现在使用 `/v1/responses` 接口而非传统的 `/v1/chat/completions`，这使得在工具调用过程中可以交叉显示推理思考。`llm` 通过 `-R` 或 `--hide-reasoning` 标志控制是否显示这些推理 token，以不同颜色区分stderr输出。

来源: [simonwillison.net](https://simonwillison.net/2026/May/12/llm/)

## 3. 开源生态 | Bambu Lab 与 AGPL 许可证争议

Jeff Geerling 发表博文强烈批评 3D 打印机制造商 Bambu Lab 滥用开源社会契约。Bambu Lab 曾向一个独立开发者发出停止令（cease & desist），禁止他维护一个允许用户绕过云服务直接使用打印机的 OrcaSlicer 分支（Bambu Lab 自己的 Linux 桌面软件采用 AGPL 许可证）。Geerling 认为 Bambu Lab"不理解开源文化"，如果公开的用户代理字符串是他们唯一的 DDoS 防护手段，那连基本的安全意识都没有。他建议用户考虑选择尊重开源原则的其他打印机厂商。

## 4. Apple 与 EU | iOS 26.5 开放第三方可穿戴设备

Apple 发布 iOS 26.5，为欧盟用户带来多项第三方设备互操作性更新：

- **接近配对**：第三方耳机可使用 AirPods 风格的快速配对
- **通知转发**：第三方智能手表可接收 iPhone 通知和 Live Activities
- **数据限制**：转发信息禁止用于广告、分析、模型训练或位置追踪

这些功能基于欧盟《数字市场法》（DMA）要求，Apple 需向第三方设备开放此前被限制的系统接口。功能仅对欧盟地区 Apple ID 用户开放，需要配件厂商主动适配。MacRumors 报道同时提到 iOS 26.5 修复了超过 50 个安全漏洞。

**来源**: [MacRumors](https://www.macrumors.com/2026/05/11/ios-26-5-eu-third-party-wearable-changes/) | [MacRumors Security](https://www.macrumors.com/2026/05/11/ios-26-5-security-fixes/)
