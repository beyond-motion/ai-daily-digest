## 视觉设计指导

### 配色方案
- 背景色: 深灰 #1a1a2e 或近黑 #0d0d14
- 主强调色: 青色 #00d4ff 或电光蓝 #0066ff
- 辅助色: 橙色 #ff6b35, 紫色 #a855f7
- 文字色: 白色 #ffffff, 浅灰 #94a3b8
- 渐变: 青→紫 对角线渐变

### 排版风格
- 标题: 大号无衬线 bold (如 Inter, Roboto)
- 数据: 等宽字体 (JetBrains Mono, Fira Code)
- 布局: 卡片式网格，多列并排
- 装饰: 细线边框，微妙发光效果

### 视觉元素
- [x] 数字高亮 (大号粗体显示关键数据)
- [x] 图标点缀 (芯片、神经网络、云架构图标)
- [x] 微网格背景 (10-20% 透明度网格线)
- [x] 发光边框 (卡片边缘微发光)
- [x] 连接线 (相关元素间的细线连接)
